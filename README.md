# 🤳Image Gallery

## Project Overview
This is a simple Image Gallery project built using HTML, CSS, and JavaScript. It displays images in a grid layout and allows users to view them in a larger popup with Next and Previous buttons.

## 🗝️Features
- Responsive image gallery
- Lightbox popup
- Next and Previous navigation
- Hover effects
- Mobile-friendly design

## 🛠️Technologies Used
- HTML
- CSS
- JavaScript
