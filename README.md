# 🖼️ Image Gallery

A responsive and interactive Image Gallery web application developed as part of the CodeAlpha Frontend Development Internship.

## 🚀 Project Overview

The Image Gallery is a responsive and interactive web application that displays a collection of images in a clean grid layout. Users can click on any image to open it in a lightbox popup and navigate through the images using **Next** and **Previous** buttons.

The gallery includes hover effects, smooth transitions, and a fully responsive design that works seamlessly on desktop, tablet, and mobile devices.

## ✨ Key Features

- 📱 Responsive image gallery layout
- 🔍 Lightbox popup for full-size image viewing
- ⬅️➡️ Next and Previous navigation buttons
- 🎨 Hover effects and smooth transitions
- 🗂️ Optional image filtering by categories
- 🖥️ Clean and user-friendly interface
